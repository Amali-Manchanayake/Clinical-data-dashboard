# Clinical Data Dashboard

## Overview
This project explores healthcare and patient service data using SQL and Power BI to identify trends, risk indicators, and operational insights.

## Objectives
- Analyse healthcare-related datasets
- Develop SQL querying skills
- Build interactive Power BI dashboards
- Visualise trends and key performance indicators

## Tools
- Power BI
- SQL
- Excel

## Project Structure
```text
data/          -> Raw datasets
sql/           -> SQL queries
powerbi/       -> Power BI dashboard files
screenshots/   -> Dashboard screenshots
docs/          -> Project documentation
```
